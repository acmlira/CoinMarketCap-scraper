# Crypto Coin Easy API
Easy API to provide up-to-date data on crypto coins.

## Setup

Install the dependencies:

```
yarn install
```

Start command:
```
yarn start
```

Open your browser in [http://localhost:3000]()

## Development setup

Start `swagger-autogen`:

```
yarn swagger-autogen
```
